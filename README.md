## Getting Started

### Prerequisites
- Python 3.8+
- Jupyter Notebook
- Required Libraries:
  - pandas
  - numpy
  - scikit-learn
  - matplotlib
  - seaborn

### Installation

1. Clone the repository
```bash
git clone https://github.com/PranikaC/Customer-Response-Prediction.git
```

2. Install required dependencies
```bash
pip install -r requirements.txt
```

## Usage

Open the Jupyter Notebook:
```bash
jupyter notebook Predicting_CustomerResponse.ipynb
```
